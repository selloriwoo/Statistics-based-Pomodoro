# 자동 시간 추천
user_sessions.json이 없으면?	기본값 1500초 (25분)이 사용됨 ? 의도된 동작임
JSON에 오늘 날짜만 있으면?	표본이 너무 적어서 추천값이 부정확할 수 있음
너무 짧거나 긴 시간만 저장돼 있으면?	표준편차에 따라 추천시간이 왜곡될 수 있음 (극단값 보정이 없음)
집중 시간 0분은 계산에서 제외하였음
